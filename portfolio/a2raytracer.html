MCG A2 Ray Tracer

A Software Based Ray Tracer built in C++

Built from the Ground Up

This is a software based ray tracer that was made for Mathematics for Computer Graphics at Bournemouth University. For every pixel on the screen, it casts a single ray out into the world and retrieves the colour value by bouncing off of different objects.

Dependencies

The project was built from the ground up using GLM for maths functions and SDL to render the window. The limitations for SDL was that it could only be accessed through a drawPixel function that would be passed a position and a colour.

Top Level Overview

When a ray is casted into the world it naively checks against every other object in the world to see if it has collided with anything. If it has hit something, it will calculate the light at that point. An additional ray is also casted towards every light source in the scene.

Once the lighting has been fully calculated, it will reflect the ray and repeat the whole process until it has reached the max bounce limit or it has run out of energy.

Optimisations

One of the major optimisations to this code would have been to add a quad-tree or some sort of bounding box hierarchy. This would have massively improved performance as a ray casted into the world would not have to check against every object.
